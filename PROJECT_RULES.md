# Règles et Contexte du Projet

## Préférences Générales
- **Langue** : Tous les documents (plans, tâches, comptes-rendus) et les échanges doivent être rédigés en **Français**.
- **Cible** : L'application est destinée à des collégiens (élèves de 5ème). Le ton doit être encourageant et adapté.

## Design et UX
- **Style** : Interface moderne, "Kawaii", ludique et motivante.
- **Thème** : Dark mode par défaut, couleurs vibrantes (gradients).
- **Interactivité** : Animations fluides, feedback immédiat (boutons X/V, toasts).

## Technique
- **Stack** : Blazor WebAssembly (PWA).
- **Persistance** : `localStorage` pour sauvegarder la progression (UserProgress).
- **Déploiement** : GitHub Pages.

## Fonctionnalités Clés
- **Fiches** : Révisions par matière/thème, cartes retournables.
- **Quiz** : Mode aléatoire ou par matière.
- **Gamification** : Système de niveaux et d'avatars évolutifs.
